Aula 03 - Variáves >

Variáveis:

São como legendas, que facilitam na hora de guardar valores e informações que vamos utilizar mais na frente no código.

Ex:
var meuNome = "Felipe";

Prompt:

Caixa de diálogo, que permite ao usuário digitasr algum dado

Ex:
var resposta = prompt("Responda essa pergunta: ");

Confirm:

Pergunta de sim ou não, devolve o valor "true" ou "false" - dependendo da resposta do usuário.

Aula 04 - Operadores Matemáticos e Tipos Primitivos >

Principais tipos primitivos:

String = tudo que está entre aspas
ex: var nome = "João";
ex: var frase = "Eu tenho 2 gatos";

Number = qualquer valor numérico ex: var meuNumero = 8;

typeof( ) = função para descobrir o tipo primitivo

Operadores matemáticos:
  • + (soma)
  • - (subtração)
  • / (divisao)
  • * (multiplicação)
  • ** (potenciação)
  • Math.sqrt() (raiz quadrada)
Operadores com variáveis:

var soma = 8 + 8
soma = 16
Com dois Numbers será feito um cálculo normal de matemática

var frase = "Oi, " + "tudo bem?"
frase = Oi, tudo bem?
Com duas strings, juntará as duas frases

var stringNumber = "Meu número é: " + 7
stringNumber = Meu número é 7
Com String e Number, juntará os dois, virando apenas uma string

Aula 07 - Laços de Repetição For & While >

Executam uma certa ação determinadas vezes

For loop

Recebe 3 parâmetros, um valor inicial, uma condição para seguir, e um salto (de quanto esse valor vai variar).

Ex:

                var i;
                for (i = 0; i < 10; i++){
                    condição
                }

While loop

Verificará a verácidade de uma condição e repetirá uma (ou mais) ação(ões) enquanto está for verdadeira

Ex:

                var i = 0;
                While (i < 10){
                    i++
                }
