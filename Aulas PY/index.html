Aula 04 - Variáveis e Print >

Variáveis:

Espaços na memória do computador que armazenam um valor

Variavel recebe valor

Como escrever em isso em Python:

variavel = "valor"

Print:

Imprime uma mensagem na tela

Em python:

Print("Mensagem")

Aula 05 - Print formatado e Input >

Print formatado:

Permite imprimir uma variável no meio de uma frase

Como formatar:

Basta adicionar um f antes das aspas e chaves ao redor de cada variável

Print(f"Mensagem {variavel} resto da mensagem")

Input:

Serve para receber dados do usuário

Para utilizar, basta escrever Input() que o computador espererá uma resposta do usuário, porém, é muito comum adicionarmos uma frase ao input para o usuário saber o que digitar.

Ex: Input("Digite algo aqui: ")

Aula 07 - Laços de Repetição For & While >

Executam uma certa ação determinadas vezes

for loop

Para uma certa variável em uma faixa, faz uma ação.

Recebe 3 parâmetros, um valor inicial, um final - obrigatórios - e um opcional, o salto, que determina de quanto em quanto esse valor vai variar.

for variavelX in range (valorInicial, valorFinal, salto)

while loop

Verificará a verácidade de uma condição e repetirá uma ação enquanto está for verdadeira

while condição:

                    ação
